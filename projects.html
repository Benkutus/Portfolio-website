Movie app

Simple movie app made with JavaScript/Node.js, shows popular movies and option to save your favorites, fetches API from TMDb.

  • Frontend built with HTML, CSS and Javascript, and handles user interactions (searches, favorites, etc.), dynamic UI rendering and communication with backend server via Fetch API. Hosted by Vercel.
  • Backend built with Node.js, Express.js, and MongoDB, handles data storage, user authentication, and API requests to fetch movie data. Hosted with Render.
  • Used libraries and frameworks: Express, mongoose, jsonwebtoken, passport, bcrypt, dotenv, cors, axios, and cookie-parser.
Personal organizer

Personal organizer for fullstack course project. Hosted locally with mongoDB as it's only for my personal use.

  • Allows you to save tasks for specific dates, which will then appear in the Tasks list, and will be updated on the dates marked in the calendar.
  • All tasks can be removed with a click of a button. Also has a feature to save notes to a different list.
  • Possible to browse different months in the calendar.
